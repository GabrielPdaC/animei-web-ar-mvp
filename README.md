# animei-web-ar-mvp — VividAR

A lightweight WebAR application that detects a visual marker through the camera and plays a video overlay in real time. Designed as a simple MVP to demonstrate image tracking and augmented video experiences directly in the browser, without requiring app installation.

The project lives in the **`vividar/`** directory. See [`vividar/README.md`](vividar/README.md) for full setup and usage instructions.
